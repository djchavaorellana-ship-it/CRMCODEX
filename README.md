# TOP CRM

Aplicacion web responsiva para seguimiento comercial de leads de TOP Event Production.

## Comandos

```bash
npm install
npm run dev
```

Para produccion:

```bash
npm run build
npm start
```

## Despliegue en Railway

1. Sube este proyecto a GitHub.
2. Crea un nuevo proyecto en Railway desde el repositorio.
3. Railway detectara Node.js.
4. Usa `npm run build` como build command y `npm start` como start command si te lo solicita.
